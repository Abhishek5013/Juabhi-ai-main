# **App Name**: RealTimeChatAI

## Core Features:

- Real-Time Chat: Real-time chat interface for immediate interaction.
- AI Chatbot: AI-powered chatbot responses based on user prompts and context awareness. The LLM will use a tool to incorporate the message history and any specific information provided in the current turn, into its reply.
- Message Display: Display user and bot messages in a clear, conversational format.
- Clear Context: Ability to clear the current conversation to reset context.
- UI Design: Simple, intuitive user interface focused on ease of use.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and intelligence.
- Background color: Light gray (#F0F0F0) for a clean, modern look.
- Accent color: Teal (#009688) to highlight important UI elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined look.
- Simple, outline-style icons for a clean, modern feel.
- Clean, spacious layout with a focus on readability and ease of use.
